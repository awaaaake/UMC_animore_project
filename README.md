# Animore - 반려동물 미용예약 서비스

Animore는 반려동물의 미용 예약을 편리하게 할 수 있도록 도와주는 서비스입니다.

## 📌 프로젝트 개요
- **프로젝트명:** Animore
- **설명:** 반려동물 미용예약 서비스
- **UMC:** 4기 프로젝트
- **기술 스택:** React

## 📌 FE팀 구성
